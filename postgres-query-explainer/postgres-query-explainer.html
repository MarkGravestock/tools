EXPLAIN, explained

starting Postgres…

A PostgreSQL query explainer that runs entirely in your browser (via PGlite, real Postgres compiled to WASM). Run a query and get its EXPLAIN ANALYZE plan back as an annotated tree: what each step does, where the time went, and where the planner guessed wrong.

Plain SQL only (CREATE TABLE + INSERT statements). For an existing database use pg_dump --inserts --no-owner --no-privileges — the default COPY format isn’t supported here.
