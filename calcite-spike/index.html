Apache Calcite in the browser — feasibility spike

Real Apache Calcite 1.41 (unmodified jars, including Janino runtime code generation) running on the CheerpJ WebAssembly JVM. First load fetches the CheerpJ runtime from its CDN plus a ~28 MB jar; both are cached by the browser.

    Try your own SQL

    Tables EMP(emp_id, name, dept_id, salary) and DEPT(dept_id, dept_name) are pre-registered from sample CSV. Unquoted identifiers are uppercased, standard Calcite lexing.
